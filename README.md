# Library Management System

A console-based Library Management System developed in C++ that manages book inventory and borrower records using file handling. The system provides an easy-to-use menu-driven interface for librarians to maintain library data, track borrowed books, and manage borrower information efficiently.

# Features
# Book Management
Enter initial book stock
View available stock
Add new books
Update book information
Delete books
View detailed book records

# Borrower Management
Register new borrowers
Update borrower information
Delete borrower records
View borrower details

# Borrowing System
Track borrowed books
Check remaining borrowing time
Maintain records using text files

# How to Compile
# Using g++:

g++ library-management.cpp -o library
How to Run
./library

# For Windows:

g++ library-management.cpp -o library.exe
library.exe
